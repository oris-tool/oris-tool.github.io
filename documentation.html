Syntax

PDF syntax

Oris accepts the following expressions:

	EXPOLY := TERM { '+' TERM }
	TERM   := NUMBER { '*' XTERM }
	XTERM  := 'x' | 'x^'INT | 'Exp[' NUMBER 'x]'
	NUMBER := any valid BigDecimal string, e.g., '-1.2'
	INT    := any positive Integer string, e.g., '2'

So, these are valid expressions:

	1
	0.5 + 0.5
	0.5 + -0.5
	0.5 + -0.5*x
	0.5 + -0.5 * x
	0.5 + -0.5 * Exp[1.0 x]
	0.5 + -0.5*x*Exp[ -1.0 x]
	0.5 + -0.5 * x * Exp[ 2 x]
	0.5 + -0.5 * x * Exp[ 2 x] * x^2
	0.5 + -0.5 * x^2 * Exp[ 2 x] + x

... but these are not:

	0.5 - 0.5
	0.5 + - 0.5
	0.5 + -0.5 * Exp[1.0 x ]
	0.5 + -0.5 * exp[1.0 x]
	(0.5 + 0.5)
